# 📱 APU Event Management Mobile Application

A cross-platform mobile application built using **Flutter** and **Firebase** that allows students to browse, explore, and stay updated with university events through a social media–style interface.

---

## 🚀 Overview

This application provides a centralized platform for students to discover events organized by different student organizations within APU. Inspired by modern social media feeds, the app delivers event content in a dynamic and user-friendly way.

---

## ✨ Features

- 📌 Browse events from various student organizations  
- 🔄 Real-time updates using Firebase Firestore  
- 🧭 Social media–style event feed for easy navigation  
- 📄 View detailed event information  
- 🏫 Explore different student organizations  

---

## 🛠️ Tech Stack

- **Frontend:** Flutter (Dart)  
- **Backend:** Firebase Firestore, Firebase Authentication, Firebase Storage

---

## 🧠 System Design

- Implemented a **NoSQL database structure** using Firestore to manage:
  - Events  
  - Student Organizations  
  - User-related data  

- Designed for **real-time synchronization**, allowing users to see updated event data instantly without refreshing.

- Structured the application with separation between:
  - UI components  
  - Data models  
  - Firestore services
